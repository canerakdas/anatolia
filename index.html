ccNetViz
ccNetViz is a lightweight, high performance javascript library for large network graphs visualization using WebGL.
It enables custom styling of nodes and edges in css like way, curve edges, dynamic changes of the network, force-directed layout and basic graph interactivity.
Used for example by Cell Collective project. ccNetViz is open source library available under GPLv3 License.
